# 📄 Documentação do Projeto EBAC Games

Este documento reúne todas as etapas do desenvolvimento do projeto **EBAC Games**, desde a pesquisa até os testes com usuários. Nele, você encontrará:

- ✅ Definição do problema e objetivos do projeto  
- 🔍 Análise de benchmarking com concorrentes diretos e indiretos  
- 👤 Criação de personas e proto-personas  
- 🗺️ Jornada do usuário (Golden Path)  
- 📝 Wireframes de baixa e média fidelidade  
- 🎨 Guia de estilo (Style Guide)  
- 💻 Protótipos navegáveis de alta fidelidade  
- 🧪 Planejamento e aplicação de testes com usuários  
- 📊 Resultados e aprendizados adquiridos  

Este material foi produzido como parte do curso da **EBAC** com foco em **UX/UI Design**, e serve como registro completo das decisões e estratégias utilizadas para a criação de uma experiência de usuário envolvente e funcional para um site de venda de jogos.
